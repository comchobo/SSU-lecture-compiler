1.	Initializer의 구현
Code generator는 간단하게 구현했고, 이후 initiailizer를 추가 구현하는 과제가 부여되어 이를 구현하였다. 구조체나 배열의 경우 해당 타입과 잘 들어맞는지, 또 해당 타입에 맞게 잘 값이 삽입됐는지, 허용되는 규칙은 무엇이 있는지 등 구현할 내용이 상당히 많아 간단하게 정수를 넣을 수 있는 초기화 기능만을 구현하였다.
우선 이를 위해 semantic.c에서 추가로 initializer 함수를 구현할 필요가 있었다. 그래야 초기화할 값을 노드에 저장할 수 있어 노드를 참고하는 것 만으로 초기화할 값을 확인할 수 있기 떄문이다.
